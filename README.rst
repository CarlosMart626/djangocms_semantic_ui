=====================
DjangoCMS Semantic UI
=====================

|pypi| |build|

Semantic UI reusable components for DjangoCMS.

Available Components
====================

- Container
- Grid
- Column
- Segments
- Group Segments
- Dividers
- Tabs

Usage
=====

Install using ``pip install djangocms-semantic-ui`` and add into installed apps inside your djangocms
project

Settings
========

.. code:: python

    INSTALLED_APPS = (
        # ...
        'djangocms-semantic-ui',
    )
